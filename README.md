# お祭りマップ＆カレンダー（世田谷・杉並・目黒・大田・川崎）

世田谷区とその隣接エリア（杉並区・目黒区・大田区）＋神奈川県川崎市のお祭りを、
「地図」と「カレンダー」で1画面から探せる、スマホ対応のWebアプリ。

- `index.html` … アプリ本体（Leaflet地図＋カレンダー＋一覧＋フィルタ）
- `festivals.js` … お祭りデータ（ここを編集すれば地図・カレンダー両方に自動反映）

ビルド不要。ブラウザで `index.html` を開くだけで動作します。

## 機能
- 地図ピン（クリックでポップアップ）とカレンダーが連動
- **エリア（区/市）** と **種類（例大祭/盆踊り/夏祭り/阿波踊り/市・縁日）** で絞り込み
- エリア・名前で年間横断検索
- 終了したお祭りは既定で非表示（トグルで開閉）

## お祭りの追加・修正
`festivals.js` の配列に1件追記するだけです。
（`name` / `area` / `place` / `lat` / `lng` / `category` / `status` / `dates` / `time` / `desc` / `url`）

- `area` … "世田谷区" / "杉並区" / "目黒区" / "大田区" / "川崎市"（省略時は世田谷区扱い）
- `status` … 日程の確度。🔴 `confirmed`（確定）/ 🟡 `annual`（例年・要確認）
