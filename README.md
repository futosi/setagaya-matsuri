# 世田谷区 お祭りマップ＆カレンダー

世田谷区のお祭りを「地図」と「カレンダー」で1画面から探せる、スマホ対応のWebアプリ。

- `index.html` … アプリ本体（Leaflet地図＋カレンダー＋一覧）
- `festivals.js` … お祭りデータ（ここを編集すれば地図・カレンダー両方に自動反映）

ビルド不要。ブラウザで `index.html` を開くだけで動作します。

## お祭りの追加・修正
`festivals.js` の配列に1件追記するだけです（名前・場所・緯度経度・日程・説明・URL）。

日程の確度は 🔴確定 / 🟡例年（要確認）の2種でバッジ表示しています。
